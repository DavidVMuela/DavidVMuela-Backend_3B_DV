# Inicializar el repositorio local
```
git init
```
- Referencia del repositorio local con el repositorio remoto (GitHub)
````
git remote add origin https://github.com/DavidVMuela/DavidVMuela-Backend_3B_DV.git


# Validar referencia
````
git remote -v
````
